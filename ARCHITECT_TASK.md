# ArchitectAgent任务描述

## 项目背景
开发**六自由度新能源车辆动力学仿真系统**，用于实时HIL测试。

## 核心需求
1. **6自由度模型**：纵向、横向、垂向、侧倾、俯仰、横摆
2. **部件解耦**：独立模块，易于替换和测试
3. **实时仿真**：满足HIL测试要求（延迟<1ms）
4. **从零实现**：学习开源，但不照搬

## 技术决策（已确认）
- **语言**：Python + C++（关键模块C++，接口Python）
- **实时**：Linux + PREEMPT_RT
- **通信**：混合（模块内共享内存，模块间消息队列）
- **精度**：默认500Hz，可配置500-1000Hz
- **参考**：CarSim架构 + OpenModelica + PyDy + 自研
